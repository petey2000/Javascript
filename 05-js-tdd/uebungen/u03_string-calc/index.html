Übung 3: String-Calculator

Tatsächlich ist der StringCalculator eine bekannte Übung von Roy Osherove. Genauer gesagt ist es eine Kata (dazu später mehr). In dieser Übung bauen Sie den String-Calculator mit seinen Specs selbst auf. Wenn Sie Probleme mit regulären Ausdrücken haben, dann gucken Sie sich erneut Kapitel 13 der Class "JavaScript: Aller Anfang ist einfach!" an.

  1. Bauen Sie den String-Calculator erst einmal nach, so dass Sie auf dem gleichen Stand sind wie Tobias und Katharina. Dazu sollten Sie folgende Specs erfüllen: 
                
              sum
  ✓ should add up all numbers from an Array

stringCalc
  ✓ should return 0 for an empty string
  ✓ should return a single number from string
  ✓ should return the sum of two comma separated numbers
  2. Jetzt kommt eine neue Anforderung dazu: stringCalc soll Zahlen größer als 1000 ignorieren. Beispiel: stringCalc('1,1001,2') // => 3
  3. stringCalc soll neben Kommata auch den Zeilenumbruch (\n) als Trennzeichen akzeptieren. Beispiel: stringCalc('1\n2,3') // => 6
  4. stringCalc soll nun wählbare Trennzeichen akzeptieren. Um das Trennzeichen zu wechseln, muss der String mit // beginnen, danach folgt das Trennzeichen und ein Zeilenumbruch, also nach dem Schema:
    Beispiel: stringCalc('//;\n1;2') // => 3
    Die erste Zeile ist optional. Alle bisherigen Specs gelten weiterhin unverändert.
