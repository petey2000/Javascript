Übung 2: Auch »Wörter zählen« will gelernt sein ...

Die Funktion countWords ermittelt die Anzahl der Wörter in einem String. Wenn Sie Probleme mit regulären Ausdrücken haben, dann gucken Sie sich erneut Kapitel 13 der Class "JavaScript: Aller Anfang ist einfach!" an.

Schreiben Sie Unit-Tests für folgende Fälle:

  • countWords('')
  • countWords('word')
  • countWords('This is a string')

Arbeitet die Funktion korrekt? Falls nein, verbessern Sie sie.
