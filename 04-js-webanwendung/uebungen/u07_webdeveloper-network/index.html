Übung 7: Dem Webbrowser beim Arbeiten zusehen

Öffnen Sie das «Network»-Tab der Developer-Console im Webbrowser, während der Browser eine REST-API benutzt (z.B. beim Lösen der vorigen Übung). Versuchen Sie, sich mit dem Werkzeug anzufreunden: Es ist eine unerlässliche Debug-Hilfe beim Arbeiten mit REST-APIs.
