Übung 6: Öffentliche APIs anzapfen

Im Web gibt es zahlreiche öffentlich zugängliche REST-Schnittstellen. Suchen Sie doch einfach mal nach »public rest api« oder nach »open data rest api«…

In dieser Übung wollen wir eine Star-Wars-API anzapfen. https://swapi.api.webmasters-akademie.dev/ Die Daten stammen aus dem Open-Source-Projekt swapi.co von Paul Hallet. Das Projekt stellt alle erdenklichen Infos aus den Star-Wars-Filmen per REST-API zum Auslesen zur Verfügung.

Unter anderem gibt es Infos zu allen Planeten, die in den Filmen vorkommen. Ein Aufruf von https://swapi.api.webmasters-akademie.dev/planets/1/ liefert etwa (siehe link)

Programmieren Sie ein kleines Frontend, das die Namen (Feld name) aller vorhandenen Planeten ausgibt.

Sie können über https://swapi.api.webmasters-akademie.dev/planets/ alle Planeten in einem Ruck einlesen.
