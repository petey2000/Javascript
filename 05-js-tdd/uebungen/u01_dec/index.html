Übung 1: Immer weniger

Schreiben Sie für die Funktion dec einen Test. dec nimmt ein Argument entgegen und gibt den Wert um 1 verringert zurück. const dec = v => v - 1

Schreiben Sie einen Unit-Test dazu und rufen Sie ihn mit Mocha auf.

Hinweise

Sie finden im Ordner exercise/01 des Begleitmaterials den Ordner less_and_less, der die vorbereitete Übung enthält. Die Datei package.json enthält Mocha und Must bereits als Abhängigkeiten. Sie können sie nachinstallieren, indem Sie im Projektverzeichnis einfach npm i eingeben.
