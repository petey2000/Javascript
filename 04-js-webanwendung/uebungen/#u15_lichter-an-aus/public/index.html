Übung 15: Lichter

Schreiben Sie eine kleine Webanwendung, die eine Glühlampe darstellt. Die Glühlampe kann entweder »an« oder »aus« sein. Die beiden Illustrationen dazu finden Sie im Verzeichnis 07/lichter:

Ein Klick auf die Illustration soll den Zustand der Lampe zwischen »an« und »aus« wechseln.

Halten Sie den aktuellen Zustand der Lampe in einer Variable auf dem Server fest und benutzen Sie WebSockets, um die Darstellung in allen verbundenen Webbrowsern zu synchronisieren.
